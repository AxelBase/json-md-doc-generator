Privacy Policy

Last updated: 2025/10/01

We respect your privacy and designed json-md-doc-generator to operate without collecting, storing, or transmitting any personal or sensitive data.

Local Processing Only

All features of this tool run entirely in your web browser. When you upload a JSON file or paste content, the data is processed in memory only. It is never sent to a remote server, saved on disk by the app, or logged for analytics. Once you refresh or close the page, all content is permanently discarded.

No Tracking

We do not use cookies, localStorage, or tracking technologies to monitor your usage. The app does not collect usage statistics, device identifiers, or personal information.

Downloaded Files

Generated Markdown files are created on your device using browser APIs. They are stored only where you choose to save them. The app has no access to your filesystem or saved documents.

Third-Party Libraries

The app uses open-source libraries for UI and functionality (such as Svelte, Bootstrap). These libraries run locally and do not transmit your data.

External Links

If the site links to third-party resources (e.g., documentation), those sites may have their own privacy policies. We are not responsible for their practices.

By using this tool, you consent to the privacy practices described above. If you do not agree, simply refrain from using the tool.
